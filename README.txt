Руководство
Приложение имеет 3 режима работы и дополнительный режим параллельных запросов (Task.Run()).
Запросы идут к https://api.weather.yandex.ru/v2/forecast
Получите API_KEY (7 дней триал) по ссылке в личном кабинете https://yandex.ru/pogoda/b2b/console/api-page и положите в файл API_KEY.txt

-В режиме прогноза погоды каждый поток получает сводку по прогнозу погоды каждые 10 сек и кеширует на 60 сек в оперативную память.
Прогноз выдается по координатам для г.Пермь на текущий день. Город прогноза погоды в данной версии изменить нельзя.
-В режиме логгирования каждый поток пишет в свой джсон файл.
-В режиме чтения из лога каждый поток читает из своего файла.
